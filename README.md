# discord_frontend

## 커밋 스타일

| 작업 유형 | 설명                                  |
| --------- | ------------------------------------- |
| feat      | 새로운 기능 추가                      |
| fix       | 버그 수정                             |
| docs      | 문서 수정                             |
| style     | 들여쓰기, 세미 콜론 등의 변경         |
| refactor  | 코드 리팩토링                         |
| test      | 테스트 코드의 작성 및 수정            |
| chore     | 외부 라이브러리 임포트 등의 작업 완료 |
